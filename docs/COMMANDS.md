# Commands and controls

The supplied notes document these in-world photo triggers:

```text
/trigger br_photo set 1   # Level 0
/trigger br_photo set 2   # chase corridor
/trigger br_photo set 3   # industrial drop area
/trigger br_photo set 4   # observation room
/trigger br_photo set 5   # pipes
/trigger br_photo set 99  # current chapter checkpoint
```

These triggers are documented for screenshot capture and checkpoint return. Their behavior has been statically inspected, not executed in a Java 1.21.5 client. Do not expose administrative commands to untrusted players.
