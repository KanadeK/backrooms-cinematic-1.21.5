# Story and gameplay (from static source review)

The data pack initializes players in a Level 0-style maze, tracks fragments and a radio gate, then routes players through a chase, fall/industrial area, observation choice, pipes, valve objectives, and final completion logic. Functions also reference tapes, anomalies, clues, a secret route, environmental flicker/footsteps/impact events, and hostile entities.

Static source inspection found several completion branches, including signal, severance, return-address and surveyors-loop paths. This is not a full-playthrough claim: event timing, room construction, combat, checkpoint behavior and each ending require real-client verification.
