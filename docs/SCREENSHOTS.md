# Visual references

This repository distinguishes client-view captures from world renders. Do not describe world renders as Minecraft F2 screenshots.

## Owner-supplied client-view captures

- `screenshots/client-view/client-view-pipes.png` — pipes/industrial client view; crosshair visible.
- `screenshots/client-view/client-view-entity-corridor.png` — corridor client view with an entity silhouette; crosshair visible.

These two files were supplied by the project owner. They are visual references, not independent proof of a full Java 1.21.5 playthrough, performance benchmark, or multiplayer test.

## World renders

The following files came from `Backrooms_v0.5.2_真实世界渲染图.zip`, supplied by the project owner. Its included README states that they were rendered directly from the uploaded world's final Anvil `.mca` files using embedded `resources.zip` textures and the v0.5.2 photo-point coordinates. They are not AI-generated and are not official Minecraft F2 captures.

- `world-renders/01_level0.png`
- `world-renders/02_chase_corridor.png`
- `world-renders/03_industrial_fall.png`
- `world-renders/04_observation.png`

For a future verified F2 capture, use normal first-person view, F1 to hide the HUD, and `br_photo` values 1–5 for the named locations. Ordinary crop/lossless compression is acceptable; do not use AI, Blender, web rendering or offline rendering to fabricate gameplay images.
