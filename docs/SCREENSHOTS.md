# Screenshots

No image in this repository is presented as a Minecraft screenshot. This environment did not launch Minecraft Java 1.21.5 with this world and integrated resource pack.

After the release blockers are resolved, capture genuine F2 images from the actual client using normal first-person view and F1 to hide the HUD. Use `br_photo` values 1–5 for Level 0, chase corridor, industrial area, observation room and pipes. Ordinary crop/lossless compression is acceptable; do not use AI, Blender, web rendering or offline rendering to fabricate gameplay images.
