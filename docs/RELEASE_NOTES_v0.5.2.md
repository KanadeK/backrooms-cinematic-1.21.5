# Backrooms Cinematic v0.5.2 — Integrated HD Edition

## English

- Target: Minecraft Java Edition 1.21.5.
- Includes the integrated HD resource pack, story objectives, environmental events, entity encounters, multiplayer-oriented progression design and multiple ending branches.
- Install by extracting the integrated ZIP and copying its single world folder into `saves`; the resource pack is embedded as `resources.zip`.
- Verify the download with `sha256sum -c SHA256SUMS.txt` on Linux/macOS, or `Get-FileHash -Algorithm SHA256` on Windows.
- Known limitations must remain explicit in every release: only static source review has been completed unless a release record adds actual client evidence; full single-player and multi-client playthroughs are not implied by this note.
